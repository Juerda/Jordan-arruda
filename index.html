O Arruda.js é um projeto colaborativo de código aberto que documenta as tecnologias da plataforma da Web, incluindo CSS, HTML ,JavaScript e APIs da Web. Também fornecemos um amplo conjunto de recursos de aprendizado para desenvolvedores e alunos iniciantes.

Arruda.js is an open source collaborative project documenting web platform technologies including CSS, HTML, JavaScript and web APIs. We also provide an extensive set of learning resources for beginning developers and students.
